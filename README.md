# Doggo Search
This is a platform to search dogs with breeds and sub-breeds. User can filter dogs on breeds basis. User can also save the dogs.
## Dependencies
- Yarn
- ReactJS
## To setup project
- Install dependencies using `yarn install` & `yarn start`
